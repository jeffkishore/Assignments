Selectors

In CSS, type, class, and ID selectors are used to target and style HTML elements in different ways.

Type Selector

A type selector applies styles to all elements of a specific type, such as <p>, <h1>, or <div>.







Class Selector

A class selector, written with a dot (.) before the class name, allows you to apply the same styles to multiple elements by giving them the same class attribute.



Id Selector

An ID selector, written with a hash (#) before the ID name, targets a single, unique element on the page. Together, these selectors give developers flexibility to control the appearance of individual elements, groups of elements, or entire types of elements efficiently.
